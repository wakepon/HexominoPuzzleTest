# 新レリックアイディア（第2弾）

> Balatroのゲームデザインを参考に、**ビルドの多様性**と**シナジー**を重視して設計。
> 各レリックは「このレリックを軸にした戦術」を生み出すことを目標とする。

## 設計原則

1. **ビルド誘導**: 各レリックが特定のプレイスタイルを促進する
2. **シナジー**: 既存レリック・パターン・シールとの組み合わせで価値が変動する
3. **トレードオフ**: 一方を強化すると他方が犠牲になる選択を生む
4. **条件の多様性**: 発動条件のバリエーションで判断の面白さを作る
5. **スケーリング**: 成長系は長期戦を、即効系は短期決戦を促す

## ビルド例（レリック組み合わせ）

### 1. 初手特化ビルド
**anchor + first_strike + overload**
→ 盤面を埋めてから初手で大量消去。1回の消去で超高スコア

### 2. パターン全積みビルド
**crown + gardener + orchestra + alchemist + 紋様の護符**
→ パターン付きブロックの価値を極限まで高める

### 3. 溜め込み爆発ビルド
**patience + tetris_rule + overload**
→ 3ターン溜めて盤面を埋めてから一気に消去

### 4. 連続消去ビルド
**glass_cannon + rensha + echo + bandaid**
→ 毎ターン消去し続けて列点を積み上げる。空振り厳禁の緊張感

### 5. 経済特化ビルド
**piggybank + investor + merchant + treasure_hunter + midas**
→ ゴールドを最大化してショップで最強装備を揃える

### 6. デッキ圧縮ビルド
**minimalist + librarian + twin + 消滅の護符**
→ デッキを最小にして同じピースを繰り返し使う

### 7. ギャンブルビルド
**gambler + luckyパターン多め + glass_cannon**
→ 高リスク高リターン。うまくいけば天文学的スコア

### 8. 大型ピース特化ビルド
**heavyweight + muscle + size_bonus_5 + size_bonus_6**
→ 大きなピースを使いこなす上級者向けビルド

### 9. シール特化ビルド
**stamp + seal_master + prism + compass_rose + 刻印の護符**
→ シールの効果を最大化するビルド

### 10. タケノコ成長ビルド
**nobi_takenoko + takenoko + copy(→nobi_takenoko) + wave**
→ 縦列のみを消し続けて倍率を永続的に上げる

## 実装優先度

| 優先度 | レリック | 理由 |
|--------|---------|------|
| **高** | piggybank, investor, merchant | 経済系は基盤として必要 |
| **高** | extra_draw, extra_hand | デッキ操作の基本 |
| **高** | featherweight, heavyweight | サイズボーナスの拡張で自然 |
| **中** | anchor, first_strike, last_stand | 乗算/加算の条件バリエーション |
| **中** | crown, stamp, alchemist | パターン/シールシナジーの軸 |
| **中** | gambler, glass_cannon | リスク/リターンの代表格 |
| **中** | minimalist, recycler | デッキ構築戦略の多様化 |
| **低** | cascade | 実装が複雑（消去→再判定ループ） |
| **低** | phoenix | ゲームフロー変更が大きい |
| **低** | fortune_teller | UI実装が必要 |
